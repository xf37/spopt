Strings for attribute names (cols of geopandas.GeoDataFrame).

n_clustersint, optional, default: 5

The number of clusters to form.

allow_move_strategyNone or AllowMoveStrategy, default: None

For a different behavior for allowing moves an AllowMoveStrategy instance can be passed as argument.

initial_labelsnumpy.ndarray or None, default: None

One-dimensional array of labels at the beginning of the algorithm. If None, then a random initial clustering will be generated.

The objective function to use.
